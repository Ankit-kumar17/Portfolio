# 🚀 Modern Interactive Developer Portfolio

A cutting-edge, highly interactive developer portfolio built to showcase modern frontend engineering skills. This visually striking layout utilizes advanced 3D hardware-accelerated physics, glassmorphic UI patterns, and intelligent scroll-tracking.

![Portfolio Preview Showcase](https://images.unsplash.com/photo-1555066931-bf19f8fd1085?q=80&w=2000&auto=format&fit=crop)

## ✨ Premium Features

- **3D Parallax Tilt Cards**: Projects, Hackathons, and Services react precisely to cursor movements in 3D-space, generating physical glare and depth using `react-parallax-tilt`.
- **Framer Motion Physics**: Scroll-linked animations seamlessly stagger, slide, and pop components into existence as they enter the viewport.
- **Glassmorphic UI**: True blurred-glass card aesthetics heavily utilizing Tailwind's backdrop filters with custom lighting boundaries.
- **Custom Hardware Cursor**: A sleek, two-part interactive cursor that tracks the mouse with simulated physics and expands on interactable elements.
- **Micro-interactions**: A continuous scroll-progress indicator, glowing text gradients, and native dark mode compatibility.
- **Serverless Communications**: A fully operational contact form connected seamlessly to Formspree APIs.

## 🛠 Tech Stack

- **Framework:** React 18 & Vite
- **Styling:** Tailwind CSS & Shadcn UI
- **Animations:** Framer Motion & Parallax Tilt
- **Icons:** Lucide React
- **Forms:** Formspree API
- **Routing:** React Router v6
- **Deployment:** Vercel CI/CD

## 💻 Getting Started locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/Ankit-kumar17/Portfolio.git
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Spin up the local development server**
   ```bash
   npm run dev
   ```
